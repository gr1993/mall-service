# 박신사 (온라인 쇼핑몰)
Spring Boot를 이용한 쇼핑몰 사이드 프로젝트 
(개인 포트폴리오 및 학습 목적으로 작성)

## 백엔드 기술

* Spring Boot 3.4.1 (JDK 17)
* Spring Security
* JPA + Querydsl
* mysql connector
* Jasypt : 설정 파일 보안 적용 (jasyptKey : mypasswordkey)
* validation : hibernate.validator


## 프론트엔드 기술

* Thymeleaf
* Bootstrap 5
* JQuery 3.7.1
* grid 라이브러리
	* simple-datatables
	* jqGrid
	* Grid.js
